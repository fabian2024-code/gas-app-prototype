# GasControl - Punto de Venta PWA 🚀

**GasControl** es una aplicación web progresiva (PWA) diseñada para la gestión eficiente de ventas y distribución de gas. Ofrece un panel de administración en tiempo real, gestión de inventario (jaulas), ranking de vendedores y un sistema de pedidos optimizado.

## ✨ Características Principales

- **Dashboard Administrativo**: Vista completa del rendimiento del negocio con gráficos dinámicos.
- **Gestión de Personal**: Sistema de turnos (Terreno/Oficina) y ranking de ventas en tiempo real.
- **Ventas PWA**: Interfaz rápida y táctil optimizada para dispositivos móviles.
- **Control de Stock**: Gestión avanzada de cilindros llenos y vacíos (Jaulas).
- **Modo Prototipo Inteligente**: Funciona incluso sin conexión o en servidores estáticos (como Netlify) mediante persistencia local.

## 🛠️ Tecnologías

- **Frontend**: HTML5, CSS3 (Glassmorphism), JavaScript (Vanilla ES6).
- **Backend**: PHP (MySQL API) - *Opcional para modo local*.
- **Base de Datos**: MySQL.
- **Iconografía**: Font Awesome 6.

## 🚀 Instalación y Uso

1. **Local (XAMPP)**:
   - Clona el repositorio en `htdocs`.
   - Importa `database.sql` en phpMyAdmin.
   - Accede vía `localhost/gas-app-prototype/index.html`.

2. **Demo Online (Netlify/Vercel)**:
   - Simplemente sube la carpeta o conecta tu repo de GitHub.
   - El sistema entrará automáticamente en **Modo Prototipo**, permitiéndote probar todas las funciones usando la memoria local de tu navegador.
